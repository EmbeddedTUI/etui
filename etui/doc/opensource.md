# Open Source Components

etui is built on the following open-source libraries. We are grateful to their authors and communities.

---

## Textual

**License:** MIT
**Repository:** <https://github.com/Textualize/textual>
**Website:** <https://textual.textualize.io>

Terminal User Interface framework that powers the entire etui UI — widgets, layout, CSS styling, and event system.

---

## pyOCD

**License:** Apache 2.0
**Repository:** <https://github.com/pyocd/pyOCD>
**Website:** <https://pyocd.io>

Python-based debug probe server and CMSIS-DAP interface used by the Probe tab to connect to ARM Cortex-M targets.

---

## pySerial

**License:** BSD
**Repository:** <https://github.com/pyserial/pyserial>
**Website:** <https://pyserial.readthedocs.io>

Serial port access library used by the Serial tab for UART communication with embedded targets.

---

## Pygments

**License:** BSD 2-Clause
**Repository:** <https://github.com/pygments/pygments>
**Website:** <https://pygments.org>

Syntax highlighting engine used in the Files tab to colorize source code.

---

## xonsh

**License:** BSD 2-Clause
**Repository:** <https://github.com/xonsh/xonsh>
**Website:** <https://xon.sh>

Python-powered shell used by the Console tab to execute commands inside the workspace.

---

## packaging

**License:** Apache 2.0 OR BSD 2-Clause
**Repository:** <https://github.com/pypa/packaging>
**Website:** <https://packaging.pypa.io>

PEP 508 requirement parsing and version handling used by the Venv tab.

---

## PyYAML

**License:** MIT
**Repository:** <https://github.com/yaml/pyyaml>
**Website:** <https://pyyaml.org>

YAML parser and emitter used by the Settings manager to read and write `~/.config/etui/settings.yaml`.

---

## Python standard library

**License:** PSF License Agreement
**Website:** <https://docs.python.org/3>

etui uses several standard library modules including `asyncio`, `pathlib`, `subprocess`, `json`, `importlib`, and `concurrent.futures`.
